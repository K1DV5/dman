To install dman on Windows, for Google Chrome:

1. Place this folder where you want to keep it from now on
2. Open Google Chrome and go to chrome://extensions
3. Enable developer mode, click Load unpacked, and select the extension folder here.
4. Copy the ID of the extension
5. Open bin/dman.exe, paste the ID and press enter.
6. Reload the extension
6. (Optional) Disable developer mode

Enjoy!
